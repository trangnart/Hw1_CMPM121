    Pay as you go with Cloud build

    This page contains information about how you pay for Cloud BuildA continuous integration service for Unity projects that automates the process of creating builds on Unity’s servers. More info
    See in Glossary    .

    Overview

    Pay-as-you-go is a type of metered service billing where you pay only for the services you use. Currently, you’re charged for build minutes based on the platform for which you are building. Manage pay-as-you-go billing from the Unity Developer Dashboard.

    For detailed pricing information, see Pay-as-you-go rates.

    Terms and commitment

    At the end of each month, you are automatically charged (fees plus applicable taxes such as VAT) for all build minutes at pay-as-you-go rates. You are not charged for storage in Cloud Build. Cloud Build has a project size limit of 40GB.

    To view your usage: 1. Sign in to the Unity Developer Dashboard. 2. Select DevOps > Cloud Build > Usage.

    You can use the Usage page to track your usage and set up a budget to manage your spending.You can cancel your plan by setting your hard cap to $0.
