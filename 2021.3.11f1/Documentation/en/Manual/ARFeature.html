    AR feature set

    Unity provides powerful ARAugmented Reality (AR) uses computer graphics or video composited on top of a live video feed to augment the view and create interaction with real and virtual objects.
    See in Glossary     tools to create rich, deeply engaging augmented reality experiences that intelligently interact with the real world. Unity’s AR feature set provides a starting point for augmented reality development, spanning from head-mounted displays to mobile.

    Getting started

    Unity’s AR feature set pre-installs the correct packages needed for AR development. Unity directly supports the following AR platforms:

    • ARCore
    • ARKit
    • Magic Leap
    • Hololens via OpenXR

    For more information on how to start developing applications for these platforms in Unity, see the documentation in the list above.

    To get started with AR Foundation and add AR features to your project, see:

    • AR Foundation documentation
    • AR Foundation Samples GitHub repository

    Configuration

    To configure your Unity Project for AR, see Using AR Foundation.

    Packages

    • AR Foundation
